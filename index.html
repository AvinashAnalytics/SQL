SQL Credit Card Analysis Project

Welcome to my project on credit card spending analysis! This repository contains SQL queries used to explore and analyze credit card transaction data. Below are the key areas of analysis:

Queries

  • Top 5 Cities with Highest Spending
  • Highest Spending Month for Each Card Type
  • Transaction Details when Cumulative Spend Reaches 1M

How to Use

  1. Download the dataset from Kaggle.
  2. Import it into SQL Server Management Studio (SSMS).
  3. Run the SQL queries from the `/queries` folder.
